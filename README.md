# v2exAutoGetCoin
v2ex自动领取每日金币奖励
# 环境依赖
1.Mac OS X EI Capitan 10.11.6

2.python 2.7.10

3.需要安装  bs4,lxml
```
sudo pip install bs4
sudo pip install lxml
```

# 使用方式
1.登录信息修改成你自己的，执行即可
